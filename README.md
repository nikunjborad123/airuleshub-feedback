# AI Rules Hub Feedback

Public feedback and issue tracking for **AI Rules Hub** — the central hub for AI coding rules.

This repository exists to make AI Rules Hub better, faster, and more developer-friendly.

---

## 🎯 Purpose

This is the official place to:

* 🐞 Report bugs
* 💡 Request features
* 🚀 Suggest improvements
* 🧠 Propose rule enhancements
* 🛠 Report CLI issues
* 📢 Share product feedback

If something feels broken, confusing, slow, or missing — open an issue.

---

## 🚀 About AI Rules Hub

AI Rules Hub is a community-driven platform for sharing and discovering AI coding rules (e.g., `.cursorrules`, AI assistant configs).

Our mission:

> Standardize, share, and optimize AI coding workflows.

We are building this with developers, not just for developers.

---

## 🧭 Before Opening an Issue

Please:

1. Search existing issues to avoid duplicates.
2. Be specific and reproducible.
3. Include screenshots or logs if relevant.
4. Mention your environment (browser, OS, CLI version, etc.).

The clearer the report, the faster we can fix it.

---

## 🐞 Bug Reports

When reporting a bug, include:

* What happened?
* What did you expect?
* Steps to reproduce
* Screenshots or console errors (if any)

---

## 💡 Feature Requests

Help us understand:

* The problem you’re trying to solve
* Why it matters
* Your proposed solution (if you have one)
* Who it benefits (individual devs, teams, CLI users, etc.)

Great feature requests focus on outcomes, not just ideas.

---

## 🛠 CLI Issues

If the issue relates to:

```bash
npx @airuleshub/cli add <slug>
```

Include:

* Node version
* Package manager (npm/pnpm/yarn)
* Full terminal output
* OS (Mac/Linux/Windows)

CLI issues are treated as high priority.

---

## 📊 What This Repo Is NOT For

* ❌ Publishing new rules (use the platform submission flow)
* ❌ Private account issues (contact support)
* ❌ Security disclosures (email us privately)
